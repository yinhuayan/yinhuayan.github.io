Github + Hexo + Next + 域名搭建可自定义前端样式的免费博客【小白版】

目的：
Github + Hexo + Next + 域名搭建可自定义前端样式的免费博客
• 超快速度： Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。
• 支持 Markdown： Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。
• 一键部署： 只需一条指令即可部署到 GitHub Pages, Heroku 或其他网站。
• 丰富的插件： Hexo 拥有强大的插件系统，安装插件可以让 Hexo 支持 Jade, CoffeeScript。

准备：
GitHub账号申请
1.新建一个仓库（new repository），并命名为YourSiteName.github.io
参考（http://cyzus.github.io/2015/06/21/github-build-blog/
注：创建仓库、发布页面只需要看到第10条）
2.用一键安装包安装nodejs环境 - http://nodejs.cn/download/
http://jingyan.baidu.com/article/a948d6515d4c850a2dcd2e18.html
一般现在nodejs包含了npm3.0以上版本，可通过npm -v直接检测。

3.安装git工具 - https://git-scm.com/downloads （安装后可以不需要配置）（参考安装说明：http://blog.csdn.net/renfufei/article/details/41647875）
4.安装TortoiseGit与配置（安装后需要配置。参考安装及配置说明：http://blog.csdn.net/renfufei/article/details/41647937）
—————————————————安装git及tortoiseGit工具后，打开githup测试是否可以克隆上传————————————————————
部署:

1. 使用npm安装hexo - 命令行$ npm install -g hexo-cli
2. 初始化框架 - 命令行$ hexo init
3. ![](http://i.imgur.com/VHfSAkw.png)    
3. 初始化完成后，可输入hexo help
4. ![](http://i.imgur.com/Z2oYWOy.png)

4. 新建文章（创建一个Hello World） - 命令行$ hexo new "Hello World" (该命令在/source/_post里添加hello-world.md文件，之后新建的文章都将存放在此目录下。)
5. 新建关于我等相关页面 - 命令行$ hexo new page "about" (该命令在/source/about里添加index.md文件)
6. 生成网站 命令行$ hexo generate 或者 $ hexo g (此时会将/source的.md文件生成到/public中，形成网站的静态文件。)
7. 使用Next主题让站点更酷炫, 下载https://github.com/iissnan/hexo-theme-next 并放入项目根目录下的themes/next (可修改该模板中的内容或替换相关图片)
8. 在source文件夹中新增CNAME,404.html等固定页面
9. 启用模板, 需要修改根目录/_config.yml配置项theme: next
10. 如显示theme: next则为成功,默认是theme: landscape(可以不用管它)
11. Ctrl+C退出debug, 正常启动服务器 - 命令行$ hexo server 或者 $ hexo s (输入http://localhost:4000即可查看网站。生成默认端口是4000)
12. 部署网站 - $ hexo deploy 或者 $ hexo d, 部署网站之前需要生成静态文件，即可以用$ hexo generate -d直接生成并部署。此时需要在_config.yml中配置你所要部署的站点：
## Docs: http://hexo.io/docs/deployment.html

deploy:
type: git
repo: git@github.com:YourRepository.git
branch: master
